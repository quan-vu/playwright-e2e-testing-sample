# playwright-e2e-testing-sample

A sample for perform end-to-end testing for modern web.

### Prerequisites

- Node.js
- pnpm
- Playwright
- TypeScript

## Tutorial

Step 1: Install Playwright

```bash
pnpm create playwright
```

Step 2: Run the test

```bash
pnpm test
```


## Getting Started

### Installation

1. Install dependencies

  ```sh
  npm install
  ```
